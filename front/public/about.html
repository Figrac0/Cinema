Введение

Система бронирования билетов в кинотеатре создана для управления кинотеатром. Это веб-приложение, рассчитанное на две основные группы пользователей: администраторов и зрителей.

Администраторы получают доступ к управлению залами, фильмами и составлению афиши. Зрители имеют возможность зарегистрироваться, авторизоваться, просматривать афишу с фильтрацией и сортировкой, а также бронировать билеты на интересующие сеансы.

Для корректной работы системы требуются определенные программные и аппаратные средства. На сервере необходима ЭВМ под управлением Windows Server 2016 или выше с установленной платформой Docker Desktop версии 3.3.0 и выше. Для клиентской части требуется ОС Windows 8, macOS 10.15 или Linux с графической средой, а также один из современных браузеров (Chrome 90+, Firefox 88+, Edge 91+, Safari 14+).

А.4 Работа с системой

А.4.1 Работа с системой в режиме администратора

Вход в систему (авторизация)

Чтобы получить доступ к интерфейсу администратора необходимо перейти по адресу http://127.0.0.1/admin/, заполнить поля для авторизации и нажать кнопку «Вход». После успешной авторизации откроется главная страница панели. В левой части экрана отображается список таблиц БД.

Просмотр данных

Чтобы увидеть список записей, необходимо щелкнуть по id нужной таблицы. В списке отобразятся все доступные записи с их основными атрибутами.

Добавление экземпляра таблицы

Чтобы добавить новую запись, необходимо нажать кнопку «Добавить» рядом с названием таблицы или в списке записей, заполнить все обязательные поля (отмечены символом *) и дополнительные при необходимости. Чтобы создать новую запись, нажмите кнопку «Сохранить».

Редактирование записи

Для редактирования нужно сначала выбрать запись из списка, щелкнув по ее id. После этого будет открыта страницы редактирования. Здесь нужно внесите необходимые изменения и сохранить их, нажав кнопку «Сохранить».

Удаление записи

Удалить запись можно, выбрав запись из списка и щелкнув на ее id. В открывшемся окне в самом низу страницы необходимо щелкнуть по кнопке «Удалить». Можно произвести удаление нескольких записей одновременно. Для этого необходимо выбрать их из списка и отметить, установив флажки (или отметив флажок в шапке таблицы для выбора всех записей) Далее необходимо нажать кнопку «Выполнить» для удаления выбранных записей.

А.4.2 Работа с системой в режиме пользователя

Чтобы получить доступ к бронированию мест необходимо войти в систему или зарегистрироваться. Для входа необходимо нажать кнопку «Вход» в верхней части сайта. Система откроет окно авторизации и регистрации.

Вход в систему (авторизация)

Для входа в систему необходимо корректно заполнить все поля формы входа и нажать кнопку «Войти». В случае успешной авторизации пользователь получает доступ ко всем функциям системы, а в шапке сайта отображается его имя.

Вход в систему (регистрация)

Если учетная запись отсутствует, необходимо нажать кнопку «Регистрация» и заполнить все поля, исправив ошибки, если они появляются. В случае успешной регистрации пользователь получает доступ ко всем функциям системы, а в шапке сайта отображаются его имя.

Выбор фильма

Выбрать фильм можно на главной странице сайта. Для удобства поиска пользователь может воспользоваться фильтрами (нажав на них и выбрав необходимый параметр), сортировками или поиском (введя в соответствующее поле название фильма).

Просмотр подробной информации о фильме

Для перехода на страницу подробной информации о фильме необходимо выбрать соответствующую карточку фильма на главной странице и кликнуть на нее. Это приведет к открытию необходимой страницы.

Бронирование мест

Процесс бронирования мест начинается с выбора подходящего сеанса. Необходимо нажать на кнопку сеанса на странице подробной информации о фильме. После нажатия на кнопку сеанса открывается интерфейс выбора мест. Пользователю предоставляется визуальная схема кинозала, где можно отметить желаемые места (но не более 5). В нижней части окна отобразятся данные о выбранных местах и кнопка оплаты с общей стоимостью билета.

Для завершения операции бронирования необходимо подтвердить выбор нажатием кнопки «Оплатить». В случае успешной оплаты отображается уведомление об успешном завершении бронирования. Пользователь может закрыть окно с помощью кнопки «».

Просмотр истории бронирования

Для просмотра истории бронирования необходимо нажать на имя пользователя в шапке сайта. После этого будет открыта страница личного кабинета, на которой отображается список ранее оформленных броней.

Просмотр справочной информации

Справочная информация находится на странице, доступ к которой осуществляется через футер сайта. После нажатия кнопки «Справочная информация» откроется страница с информацией о разработчиках и кнопкой для перехода на страницу информации о системе.
